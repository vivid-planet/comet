# @comet/admin

## Development

### Requirements

-   [nvm](https://github.com/nvm-sh/nvm)

Run `install.sh` once.

### Run build and storybook

```
yarn start
```

### Develop in an project

#### additional Requirements

-   [watchman](https://facebook.github.io/watchman/)

###

    ./wml-add.sh ../example
    wml start
