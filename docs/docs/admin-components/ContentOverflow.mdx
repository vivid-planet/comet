# ContentOverflow

Use `ContentOverflow` to wrap content that may be too large to fit its container. <br />
If the content is too large, it will be truncated. When clicked, the entire content will be displayed in a dialog.

```tsx
<ContentOverflow>{/* Lots of content ... */}</ContentOverflow>
```

## Use Cases

A common use-case would be to show rich-text content in a `DataGrid`. <br />
If used outside a `DataGrid`, the height must be limited by a parent container, e.g.:

```tsx
<Box sx={{ height: 100 }}>
    <ContentOverflow>{/* Lots of content ... */}</ContentOverflow>
</Box>
```
