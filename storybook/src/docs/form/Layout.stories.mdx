# Form Layout

Different types of layouts can be achieved, by adding `variant="horizontal"`, `variant="vertical"` or `fullWidth` to your field's props.

When using `variant="vertical"` (which is the default value) the label is shown above the field. This is usually the most compact layout type.  
With `variant="horizontal"` the label is shown on the left side of the field, where all labels have a fixed width.  
Adding the `fullWidth` prop will cause the field and it's input to take up all the width, given by the parent element.

## Examples for the different types of layouts, supported by comet-admin

### Inline Fields

By default, fields are rendered next to each other and the width will be determined by the component used within the field.  
To make fields have a specific width, use the `fullWidth` prop on the field and make sure the parent-element of the fields has the width you want.
One way to do this is by using a `Grid` which can be used to have uniform size and spacing for all fields.

### Full-Width Layout

[Field](/docs/docs-form-components-field--page) or [FieldContainer](/docs/docs-form-components-fieldcontainer--page) components used within a dialog or sidebar should generally use the full-width layout.  
This can be achieved by adding the `fullWidth` prop to the fields.

```tsx
<Field name="text" label="Text" component={FinalFormInput} fullWidth />
```

```tsx
<FieldContainer label="Text" fullWidth>
    <InputBase onChange={onChange} value={value} />
</FieldContainer>
```

When using the full-width layout with larger, more complex forms, it makes sense to use [FormSection](/docs/docs-form-components-formsection--page) to group together some fields within the form.

```tsx
<FormSection title="Personal information">
    <Field name="name" label="Full name" component={FinalFormInput} fullWidth />
    <Field name="email" label="Email" component={FinalFormInput} fullWidth />
    <Field name="phone" label="Phone" component={FinalFormInput} fullWidth />
</FormSection>
```

### Horizontal layout

If you want to use the horizontal layout on an individual [Field](/docs/docs-form-components-field--page) or [FieldContainer](/docs/docs-form-components-fieldcontainer--page), you can set the `variant` prop to "horizontal":

```tsx
<Field name="text" label="Text" component={FinalFormInput} variant="horizontal" />
```

```tsx
<FieldContainer label="Text" variant="horizontal">
    <InputBase onChange={onChange} value={value} />
</FieldContainer>
```

To use the horizontal layout throughout your application, set it as the default value in your theme, on `CometAdminFormFieldContainer`.

```ts
const theme = createCometTheme({
    components: {
        CometAdminFormFieldContainer: {
            defaultProps: {
                variant: "horizontal",
            },
        },
    },
});
```
