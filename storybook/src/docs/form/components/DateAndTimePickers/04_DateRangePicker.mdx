# Date-Range Picker

To use the `DateRangePicker` and `FinalFormDateRangePicker` components, install the `@comet/admin-date-time` package.

## Value type

The `value` prop and the value returned by `onChange` are objects with `start` and `end` values, each as a string in the format `YYYY-MM-DD`.

```ts
type DateRange = {
    start: string;
    end: string;
};
```
