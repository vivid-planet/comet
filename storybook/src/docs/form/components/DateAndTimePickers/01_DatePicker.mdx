# DatePicker

`DatePicker` is a wrapper around the `DatePicker` from `@mui/x-date-pickers` to make it more consistent with other form components.  
See the [MUI docs](https://v7.mui.com/x/react-date-pickers/date-picker/) for more information.

`DatePickerField` is a further extension to allow usage inside a `FinalForm` form.
