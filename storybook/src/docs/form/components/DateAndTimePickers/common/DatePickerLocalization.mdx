## Localization

The date and time pickers build on MUI X Date Pickers and are localized through MUI's `LocalizationProvider` using the `AdapterDateFns` adapter with the desired locale. <br />
Generally, this is done at the entry point of the application, e.g., in `App.tsx`:

<Source
    language="tsx"
    code={dedent`
        import { LocalizationProvider } from "@mui/x-date-pickers";
        import { AdapterDateFns } from "@mui/x-date-pickers/AdapterDateFns";
        import { enUS as dateFnsEnLocale } from "date-fns/locale";
        export const MyApp = () => (
            <LocalizationProvider dateAdapter={AdapterDateFns} adapterLocale={dateFnsEnLocale}>
                {/* App content */}
            </LocalizationProvider>
        );
    `}
/>
