# Time Picker

Import the `TimePicker` and `TimePickerField` components from `@comet/admin`.

By default, there is only one selectable time option every 15 minutes. <br />
This shortens the list of options to reduce the need for scrolling. <br />
Use the `minutesStep` prop to change the step size.

The range of selectable options can be limited by passing `Date` objects into the `minTime` and `maxTime` props.

## Value type

The `value` prop and the value returned by `onChange` are 24-hour strings formatted as `HH:MM`, similar to the [HTML time input](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/input/time).
