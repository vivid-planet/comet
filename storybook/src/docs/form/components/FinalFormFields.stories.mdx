# Comet Admin FinalForm Fields

Comet Admin provides a set of FinalForm compatible form fields.

## FinalFormInput

`FinalFormInput` is a plain input field.
You can use it like the HTML input element.

<Canvas>
    <Story id="stories-form-finalform-fields--finalforminput" />
</Canvas>

## FinalFormSearchTextField

`FinalFormSearchTextField` is a prestyled version of [FinalFormInput](#finalforminput).
You can use it for implementing a search field.
It doesn't have any built-in search functionality.

<Canvas>
    <Story id="stories-form-finalform-fields--finalformsearchtextfield" />
</Canvas>

## FinalFormSelect

`FinalFormSelect` provides a select with multiple options.
It's possible to load the options asynchronously using the `useAsyncOptionsProps()` hook.
It expects the value to be an object to be able to display the current value without having to load the async options.

<Canvas>
    <Story id="stories-form-finalform-fields--finalformselect" />
</Canvas>

## FinalFormAutocomplete

`FinalFormAutocomplete` provides a select with a search field.
It can be used similarly to [FinalFormSelect](#finalformselect).

<Canvas>
    <Story id="stories-form-finalform-fields--finalformautocomplete" />

## FinalFormCheckbox

`FinalFormCheckbox` provides a checkbox.
It should be used inside a [Material UI `FormControlLabel`](https://v4.mui.com/api/form-control-label/#formcontrollabel-api).

<Canvas>
    <Story id="stories-form-finalform-fields--finalformcheckbox" />
</Canvas>

## FinalFormSwitch

`FinalFormSwitch` provides a switch.
It should be used inside a [Material UI `FormControlLabel`](https://v4.mui.com/api/form-control-label/#formcontrollabel-api).

<Canvas>
    <Story id="stories-form-finalform-fields--finalformswitch" />
</Canvas>

## FinalFormRadio

`FinalFormRadio` provides a radio control.
It should be used inside a [Material UI `FormControlLabel`](https://v4.mui.com/api/form-control-label/#formcontrollabel-api).

Each radio option must be a separate `Field`.
All the Fields should be wrapped in a `FieldContainer`.

    <Story id="stories-form-finalform-fields--finalformradio" />
</Canvas>

## FinalFormRangeInput

`FinalFormRangeInput` adds a range selector.
It consists of two text input fields and a slider.
This range input can, for example, be used for filtering products by price.
