# PrettyBytes

<Meta title="Docs/Components/PrettyBytes" />

The PrettyBytes component can be used to display file sizes or other byte values.

Supported units are: `"byte" | "kilobyte" | "megabyte" | "gigabyte" | "terabyte" | "petabyte"`
